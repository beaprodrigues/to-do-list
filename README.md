# Lista de Tarefas

Estruturei a Lista de Tarefas com HTML5, CSS3 e Javascript.
Criei uma estrutura básica com um input para adicionar tarefas e uma lista dinâmica para exibi-lás, o código também cria um novo item na lista. Além disso, é possível deletar tarefas ao cliclar no botão.
Estilizei o código com CSS3, desing simples moderno e a fonte importada do Google Fonts.
