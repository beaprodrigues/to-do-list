# Lista de Tarefas - Javascript

![Screen recording 1 (online-video-cutter com)](https://github.com/user-attachments/assets/ef5513db-cf24-47f2-a2d7-87c6674dfdf2)

Estruturei a Lista de Tarefas com HTML5, CSS3 e Javascript.
Criei uma estrutura básica com um input para adicionar tarefas e uma lista dinâmica para exibi-lás, o código também cria um novo item na lista. Além disso, é possível deletar tarefas ao cliclar no botão.
Estilizei o código com CSS3, desing simples moderno e a fonte importada do Google Fonts. 

## Tecnologias Utilizadas

* **HTML5**: Estrutura semântica.
* **CSS3**: Estilização e efeitos visuais.
* **JavaScript**: Interação.
* **Google Fonts**: Fonte "Poppins" utilizada.


## Funcionalidades

O projeto utiliza o Javascript para criar uma lista dinâmica para ser usada no dia a dia.

- Possível adcionar e remover tarefas manualmente.
